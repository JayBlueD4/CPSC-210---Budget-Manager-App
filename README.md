# Budget Manager App

This application can be for anyone who wants to create a budget. It was inspired by a personal desire to better manage finances while only needing to use a single application. Relevant tasks such as income tax calculations and purchase projections can be efficiently conducted without needing to switch to a separate application, like other budget applications.

Java was used to develop both the front-end and back-end of this project. Technologies learned for the development process include Java Swing for the GUI, JSON files for data persistence, and JUnit for testing.

This app is used for my personal budgeting today. I continue to research new technologies, such as XML and servers, which will enable the implementation of new features (e.g. users and a more secure login system, better design) that will improve user security and usability.
