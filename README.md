# Budget Manager App #

* Inspired by a personal desire to start managing finances. Realized that most budget managers require the use of separate applets for relevant tasks like tax calculation, purchase projections, etc., and thus wanted to make one that had everything in one place.
* This app uses Java for both back-end and front-end design.
* Technologies learned for this project include Java Swing for the GUI, JSON files for data persistence, JUnit for testing, and Java Events.
* Using this app for personal budgeting today. Continuing to research new technologies (e.g. XML, servers) to enable new features (e.g. users & more secure login system, better layout).
